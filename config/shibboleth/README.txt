CKANEXT-REMS Shibboleth settings
=================

This folder contains the example additions to Shibboleth config used by
ckanext-rems plug-in.

Add the content of shibboleth2.xml file to your shibboleth2.xml config. Add
your-rems-metadata-file.xml to the same folder with your shibboleth2.xml.
Modify your-rems-metadata-file.xml to your preferences.
